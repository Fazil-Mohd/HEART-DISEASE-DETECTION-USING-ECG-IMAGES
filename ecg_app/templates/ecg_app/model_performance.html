Architecture

ResNet50 backbone pre-trained on ImageNet with a custom dense head (GlobalAvgPool → Dense 256 + Dropout 0.5 → Dense 128 + Dropout 0.3 → Softmax 4-class). Focal Loss (γ=2.0) with label smoothing (0.05) was used to handle class imbalance.

📊
Training Strategy

Two-phase training: Phase 1 froze the backbone and trained the head for {{ phase1_epochs }} epochs. Phase 2 unfroze all layers for full fine-tuning over {{ phase2_epochs }} epochs with a lower learning rate, achieving the best validation accuracy of {{ overall_accuracy }}%.
